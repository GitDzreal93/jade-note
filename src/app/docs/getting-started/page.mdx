# 快速开始

欢迎使用 Jade Note！本指南将帮助你快速了解和使用我们的文档系统。

## 特性介绍

- 基于 Markdown/MDX 的文档编写
- 自动生成目录和导航
- 会员内容解锁机制
- 响应式设计，支持多端访问

## 文档编写

### Markdown 基础语法

```markdown
# 一级标题
## 二级标题

- 列表项 1
- 列表项 2

1. 有序列表 1
2. 有序列表 2

> 引用文本

**加粗** *斜体*
```

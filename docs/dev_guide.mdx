# 开发指南

本文档提供了 Jade Note 项目的开发指南，帮助开发者更好地理解和参与项目开发。

## 主要内容

- [项目结构](/docs/dev_guide/project_structure) - 了解项目的文件组织和架构设计
- [开发规范](/docs/dev_guide/coding_standards) - 掌握团队的代码规范和开发流程

## 技术栈

我们的项目使用了以下主要技术：

- **Next.js 14** - 最新的 React 框架，支持 React Server Components
- **TypeScript** - 为 JavaScript 添加类型支持，提高代码质量
- **Tailwind CSS** - 现代化的原子级 CSS 框架
- **MDX** - 支持在 Markdown 中使用 React 组件

## 项目特点

1. 文档管理
   - 支持 MDX 格式，可以在文档中嵌入 React 组件
   - 多级文档结构，易于组织和管理
   - 自动生成文档目录

2. 现代化开发
   - 使用 React Server Components 提升性能
   - 采用 TypeScript 确保代码质量
   - 集成 Tailwind CSS 实现响应式设计

3. 用户体验
   - 美观的界面设计
   - 快速的页面加载
   - 便捷的文档搜索

## 开发流程

1. 本地开发

   ```bash
   # 启动开发服务器
   pnpm dev
   
   # 构建项目
   pnpm build
   ```

2. 代码规范
   - 使用 ESLint 和 Prettier 维护代码质量
   - 遵循 TypeScript 类型检查
   - 编写清晰的代码注释

# 最佳实践

本文档总结了 Jade Note 项目的最佳实践和开发经验，帮助开发者提高开发效率和代码质量。

## React Server Components

1. **服务端渲染**
   - 默认使用服务端组件
   - 减少客户端 JavaScript 体积
   - 提升首屏加载速度

2. **数据获取**
   - 直接在服务端获取数据
   - 避免客户端缓存
   - 减少数据获取延迟

3. **交互组件**
   - 仅在需要时使用客户端组件
   - 使用 'use client' 指令
   - 保持交互组件的精简

## MDX 最佳实践

1. **文档组织**
   ```mdx
   # 标题

   简洁的概述。

   ## 子标题

   详细内容。

   ### 代码示例

   ```tsx
   // 精简的示例代码
   ```
   ```

2. **内容组织**
   - 使用清晰的标题层级
   - 添加适当的代码示例
   - 包含相关文档链接

3. **可访问性**
   - 使用语义化标记
   - 提供替代文本
   - 保持文档结构清晰

## 性能优化

1. **服务端渲染**
   ```tsx
   // 在服务端获取和处理数据
   async function DocPage({ params }: Props) {
     const doc = await getDocContent(params.slug);
     return <DocContent content={doc} />;
   }
   ```

2. **图片优化**
   ```tsx
   // 使用 Next.js 的图片优化
   import Image from 'next/image';

   function DocImage({ src, alt }) {
     return (
       <Image
         src={src}
         alt={alt}
         width={800}
         height={400}
         placeholder="blur"
         loading="lazy"
       />
     );
   }
   ```

3. **字体优化**
   ```tsx
   // 使用系统字体栈
   const typography = {
     sans: ['ui-sans-serif', 'system-ui', 'sans-serif'],
     mono: ['ui-monospace', 'monospace'],
   };
   ```

## 响应式设计

1. **断点设置**
   ```tsx
   // Tailwind CSS 断点
   <div className="
     w-full          // 移动端
     md:w-3/4       // 平板
     lg:w-2/3       // 桌面端
     xl:w-1/2       // 大屏幕
   ">
   ```

2. **内容适配**
   ```tsx
   // 响应式文本大小
   <article className="
     prose         // 基础样式
     sm:prose-sm   // 小屏幕
     lg:prose-lg   // 大屏幕
   ">
   ```

## 错误处理

1. **优雅降级**
   ```tsx
   try {
     const doc = await getDocContent(slug);
     return doc ? <DocContent doc={doc} /> : <DocNotFound />;
   } catch (error) {
     console.error('Error loading doc:', error);
     return <DocError error={error} />;
   }
   ```

2. **加载状态**
   ```tsx
   // 使用骨架屏
   function DocSkeleton() {
     return (
       <div className="animate-pulse">
         <div className="h-8 bg-gray-200 rounded w-1/2 mb-4" />
         <div className="h-4 bg-gray-200 rounded mb-2" />
         <div className="h-4 bg-gray-200 rounded w-3/4" />
       </div>
     );
   }
   ```
